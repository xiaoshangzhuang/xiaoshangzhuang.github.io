在linux系统上部署web项目
一．在linux系统上安装tomcat
1.下载tomcat
1.1进入 Apache Tomcat官网，选择要下载的版本。
1.2把tomcat安装包上传到服务器 /usr/local 下。

2.解压安装tomcat
2.1用Xshell等Linux远程连接工具访问服务器，进入文件夹，指令是cd /usr/local。
2.2解压安装包，指令是unzip apache-tomcat-7.0.78.tar.gz。
2.3进入bin目录，指令是cd /usr/local/tapache-omcat-7-0-78/bin。
2.4启动tomcat，指令是sh startup.sh。
2.5在ogs下查看启动日志，指令是tail -f catalina.out，可以查看到tomcat的运行信息，如果看到INFO: Server startup in 1925 ms，表示tomcat已经启动成功了，按ctrl+c退出该日志。
2.6在浏览器输入服务器的IP+端口（默认8080），能看到如下页面，这是tomcat没有部署项目的默认页面。

2.7停止运行tomcat，指令是sh shutdown.sh。

二．在linux系统上搭建java环境
1.在Centos下使用yum安装java环境。
2.查看yum库的java安装包，指令是 yum -y list java*。

3.选择合适的版本，使用yum安装。

4.查看结果，指令是 java -version，如果有java版本就表示安装成功。

三．在linux系统上安装nginx
1.下载nginx
1.1进入nginx官网，选择要下载的版本。
1.2把nginx安装包上传到服务器 /usr/local 目录下。
2.解压安装nginx
2.1解压安装之前，先检查是否安装pcre，如果没有，就用yum 安 装。命令是，yum -y install pcre 和 yum -y install pcre-devel。找 到 pcre位置，命令是 rpm -ql pcre 和 rpm -ql pcre-devel。
2.2解压nginx，命令是tar -zxvf nginx-1.10.2.tar.gz，进入 目 录，命令是，cd/usr/local/nginx-1-10-2，执行命令 ./configure。

如果这一步出错了，根据错误提示信息解决问题后，使用 make clean 命令，清除产生的临时文件，再来重新执行 ./configure
2.3编译和安装
执行命令，make 和 make install，到此安装完毕。
2.4修改nginx配置文件，命令是，vi /conf/nginx.conf。修改端口为80。
2.5启动nginx，命令是，./nginx -c。在浏览器里输入服务器ip或域名，如下图，表示成功。
